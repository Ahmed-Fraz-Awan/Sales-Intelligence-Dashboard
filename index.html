üìä Project Description

This is an interactive sales intelligence dashboard that uses machine learning to provide:

  • ‚úÖ AI Sales Forecasting - Predicts next 6 months sales using Random Forest
  • ‚úÖ Customer Segmentation - Groups customers using RFM analysis and K-Means
  • ‚úÖ Interactive Visualizations - 12+ chart types including heatmaps, treemaps
  • ‚úÖ Real-time Analytics - Filter by date, category, region
  • ‚úÖ Business Insights - AI-generated recommendations
  • ‚úÖ Modern UI/UX - Glassmorphism design with responsive layout

üìÅ Dataset Information

üìä Comprehensive Synthetic E-commerce Dataset

Source: Kaggle - imranalishahh/comprehensive-synthetic-e-commerce-dataset

3,000+ Transactions
3 Years Time Period
12+ Features
3 Categories

Dataset Features:

  • Order Date - Timestamp of transactions (2022-2024)
  • Customer ID - Unique customer identifiers
  • Categories - Furniture, Technology, Office Supplies
  • Regions - East, West, Central, South
  • Customer Segments - Consumer, Corporate, Home Office
  • Sales Amount - Transaction values with realistic ranges
  • Profit - Calculated profit margins by category
  • Quantity - Number of items purchased
  • Discount - Applied discounts (0%, 10%, 15%, 20%)
  • Seasonal Patterns - Realistic holiday peaks and summer dips
