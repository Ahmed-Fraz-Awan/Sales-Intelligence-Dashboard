Project Description

This is an advanced, interactive sales intelligence dashboard powered by machine learning that provides:

  • AI Sales Forecasting using Random Forest Regression
  • Customer Segmentation with RFM analysis and K-Means clustering
  • 12+ Interactive Visualizations including heatmaps, treemaps, and funnel charts
  • Real-time Filtering by date, category, and region
  • AI-Generated Insights with business recommendations

The dashboard helps businesses understand sales patterns, predict future trends, and make data-driven decisions.

Dataset Description

Synthetic E-commerce Sales Dataset

  • Time Period: January 2022 - December 2024 (3 years)
  • Records: 3,000+ sales transactions
  • Features Included:
    • Order Date, Customer ID
    • Category (Furniture, Technology, Office Supplies)
    • Region (East, West, Central, South)
    • Customer Segment (Consumer, Corporate, Home Office)
    • Sales Amount, Profit, Quantity, Discount
  • Special Features:
    • Seasonal patterns (holiday peaks, summer dips)
    • Realistic profit margins by category
    • Customer behavior simulation
